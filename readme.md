# CV

Karl Oskar Anderson resume in PDF friendly format. Made with ❤️ using basic HTML.

### Running the server
```
python pyServer.py
OR
python3 pyServer.py
```

### Saving to PDF
1. Open in browser
2. Open `Print` menu (`Ctrl + p`)
3. Change `Destination` to: `Microsoft Print to PDF` or `Save as PDF`
4. Click `Save` or `Print`
